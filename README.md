# Calculadora em PHP

Este projeto é uma calculadora simples desenvolvida utilizando PHP, HTML, CSS e JavaScript.

## Funcionalidades

A calculadora realiza as seguintes operações matemáticas:

- Adição
- Subtração
- Multiplicação
- Divisão
- Potência
- Radiciação

## Tecnologias utilizadas

- PHP
- HTML
- CSS
- JavaScript
